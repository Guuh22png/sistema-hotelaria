# Sistema de Hotelaria

## Descrição

Projeto de banco de dados desenvolvido para gerenciamento de hotéis, quartos, reservas, funcionários, pagamentos e avaliações.

## Estrutura do Banco

### Tabelas

- HOTEIS
- QUARTOS
- RESERVAS
- FUNCIONARIOS
- PAGAMENTOS
- AVALIACOES

## Tecnologias Utilizadas

- SQLite
- SQL

## Funcionalidades

- Cadastro de hotéis
- Gerenciamento de quartos
- Controle de reservas
- Registro de pagamentos
- Avaliações de clientes
- Controle de funcionários
